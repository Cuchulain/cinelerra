0) compile ./po/xlat.C -> ./po/xlat:  cd po; c++ xlat.C -o xlat; cd ..
1) run: ./po/xlat.sh > /tmp/xlat.po
2) run xlat.C as: ./po/xlat < /dev/null /tmp/xlat.po > /tmp/xlat.dat
3) open browser window, url = file:///tmp/xlat.dat
4) open browser window, access google -> translate, and set desired target <lang>
5) open ooffice and create empty spreadsheet
6) select all xlat.dat text in url = file:///tmp/xlat.dat, btn3 select, btn3 copy
7) paste into col A of sheet, ptr over col A title area, btn 3, paste
  option menu:  utf-8, <chk>sep by <chk>comma, Text ", <chk>quoted fields text
8) paste into google xlat input textbox, run xlat
9) at bottom of xlat output textbox, press select all tool button
10) paste into col B of spreadsheet, ptr over col B title area, btn 3, paste
11) in spreadsheet, save file as <lang>.csv file, File->save_as, <lang> type = csv
  option menu: utf-8, seperator=, text=", DONT quote text
12) edit <lang>.csv: fix probs with <lang>.csv due to auto format, eg ":"->": ",...
13) find locale identifier for desired lang == <xx> (maybe in ./po/locale_list.txt)
14) run xlat.C again, this time as: ./po/xlat < <lang>.csv /tmp/xlat.po ./po/<xx>.po
15) make sure po in targeted in ./po/Makefile

example: (french)  export LANG=fr_FR.UTF-8
example: (english) export LANG=en_US.UTF-8
./cinelerra
